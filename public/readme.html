🚀 Zippy Streetwear Platform

📋 Project Overview

Zippy is a modern, full-stack e-commerce platform for streetwear enthusiasts. Built with Node.js backend and vanilla JavaScript frontend, it features a futuristic cyberpunk design with neon aesthetics.

🛠 Technology Stack

  • Backend: Node.js with Express.js
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Database: File-based JSON storage via persist_module
  • Authentication: JWT tokens with bcrypt password hashing
  • Design: Cyberpunk theme with neon colors and glass-morphism effects

✨ Key Features

Authentication & User Management

  • Secure registration and login system
  • Remember Me functionality (12 days vs 30 minutes)
  • Default admin account: admin/admin
  • JWT-based session management

Product Management

  • Browse products with search functionality
  • Product filtering by category and brand
  • Detailed product pages with images
  • Admin product CRUD operations

Shopping Features

  • Shopping cart with add/remove functionality
  • Checkout process with mock payment
  • Order history tracking
  • Purchase confirmation emails (simulated)

Community Features

  • User exchange marketplace
  • Community forum and discussions
  • User profiles and ratings
  • Brand partnerships showcase

Admin Dashboard

  • User activity monitoring
  • Product inventory management
  • Order processing
  • Activity logs with filtering

🏗 Architecture

Backend Structure

modules/
├── auth-server.js       # Authentication routes
├── products-server.js   # Product management
├── cart-server.js       # Shopping cart logic
├── payment-server.js    # Payment processing
├── exchange-server.js   # Exchange marketplace
├── admin-server.js      # Admin functionality
└── persist_module.js    # Data persistence layer

Frontend Structure

public/
├── scripts/
│   ├── auth.js         # Authentication logic
│   ├── products.js     # Product browsing
│   ├── cart.js         # Cart management
│   └── ...
├── styles/
│   └── main.css        # Global styles
└── *.html              # Page templates

🚀 Getting Started

Installation

# Clone the repository
git clone [repository-url]

# Install dependencies
npm install

# Start the server
npm start

Environment Setup

The server runs on port 3000 by default. You can change this in the server.js file.

🧪 Testing

Run the test suite with:

node test.js

This will test all API endpoints and verify functionality.
