üíæ Data Persistence

All user data is persisted in JSON files in the file system:

  • User Details: Stored in data/users.json
  • Cart Data: Stored in data/carts.json
  • Purchase History: Stored in data/orders.json
  • Login Activity: Stored in data/activity.json
  • Products: Stored in data/products.json
  • Exchanges: Stored in data/exchanges.json

Data persists across server restarts and is loaded on startup.
